# News Site

# AI TECH TIMES 🚀
A modern, responsive news portal focusing on the latest breakthroughs in Artificial Intelligence. 

This project was built as my first assignment at Technigo to master the fundamentals of layout and styling. 

## Live Demo 
You can view the live site here: [AI Tech News Site] (https://lofstromfelicia.github.io/AI-tech-news-site-project-felicia/
)

## 📋 Project Goals
The objective was to create a mobile-first, responsive news site using: 
* **Semantic HTML5** for accessibility and structure.
* **CSS Grid** for the news article layout.
* **Flexbox** for the header and navigation. 

## ✨ Features
* **Responsive Design** Optimized for mobile (1 column), tablet (2 columns), and desktop (4 columns). 
* **Interactive UI:** Custom hover effects and transitions for a better user experience. 
* **Modern Tech Aesthetic:** A selected color palette and typography using Google Fonts (Space Grotesk & Inter).
* **Sticky Navigation:** Header stays accessible while scrolling. 

## 🛠️ Tools Used
* **VS Code** - Code editor. 
* **WSL/Ubuntu** - Terminal and environment. 
* **Git & GitHub** - Version control.
* **Google Fonts & Unsplash** - Visual assets. 

## 💡 Learning Reflections 
During this project, I deepened my understanding of the 'Mobile First' approach and the importance of writing clean, maintainable CSS using the DRY (Don't Repeat Yourself) principle.
